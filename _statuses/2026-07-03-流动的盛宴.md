# 流动的盛宴

我之前卖过一段 toC 耐用品，这种产品有一个特点，就是需要持续的大量的新客户。耐用消费品购买频次非常低，靠新客户销售才能维持经营。老客户只能是锦上添花，拉新也成了是运营工作的关键。

剥离掉房地产身上所有的金融衍生属性，在这方面和 toC 耐用消费品是一致的，正常情况下经营销售无法指望老客户今年买后明年又买。除非那种投机性非常强的时期，投资客会高频次购买。长期看新客的购买力，才是维持这个庞大系统转动的燃料。

之前的时期 2017 之前的房地产拥有最完美的外部环境：
* **工业化**提供了大量就业岗位，产生了大量的购买力。
* 国家快速的**城镇化**将数以亿计的人口推向房地产市场。
* 中央的**地产政策导向** + 满足地方政府**土地财政需求**。
* **拆迁致富、炒房致富**的神话点燃整个社会流动性杠杆火苗，居民的债务表成了政府部门和相关利益人的货币提款机和借条抵押。

天下没有不散的筵席。当宾客散尽，杯盘狼藉，账单总要有人来结。

历史上看空房地产的人：
* **2007 年**喊狼来了，要面对美国的量化宽松 + 中国的财政放水；
* **2014 年**又喊狼来了，要面对天量棚改 + 涨价去库存 + 拉到极致的居民杠杆；
* **2020 年**再次狼来了，也没人信了。

市场上充斥着傲慢的声音：
> “中国房地产怎么可能会跌？”
> “中国楼市是政策市，政府让涨就涨，让跌就跌，政策万能论。”
> “说房价跌的，都是买不起房的穷人。”

### 谁在支付账单？

居民部门的债务水平从 **25% 升到 63%**，民营企业部门的利润率从 **8% 到 3%**，很多民营企业基层员工的周实际平均工作时长可能在 **50 个小时**左右，这些都是账单的一部分。

当后工业时代，约束条件全部调转了方向：
1. **后现代制造业**无法带来净增就业岗位。
2. **老龄化**的城镇也无法再扩张城市边界。
3. **供需两端**共同走向衰弱。
4. 当中央对地产政策进行了重新审视，地方财政的再平衡和**土地财政难以为继**。

一切的约束条件都朝另一个方向了，房地产的中场已经有了结论。

一代人 20 多岁遇到就业危机和收入困难，职业生涯还未起步就已经结束。下半场的盛衰如何，那是能参与下半场的人的事情，与上半场的多数人已经无关。
